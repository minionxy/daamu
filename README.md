# Achu & Damu 💗

A little love-letter website from **Achu** to **Sandra** — the girl everyone calls *Damu*, and the one only he gets to call *Paru*. Built with plain HTML, CSS, and JavaScript. No framework, no build step. Just open `index.html` in a browser, or upload the folder to GitHub Pages.

## What's inside
- 💖 Animated intro with floating hearts
- 📖 Our story timeline (Onam 2025 → Oct 19 → Oct 22 → forever)
- ✨ Rotating love quotes
- 🖼️ Photo gallery (click to zoom)
- 🎮 Mini game: scratch 6 cards to reveal hidden memories
- 💌 A handwritten-style letter
- 🎵 Background music toggle (top right)

## How to publish on GitHub Pages
1. Create a new GitHub repository (e.g. `daamu`).
2. Upload **all files in this folder** (`index.html`, `style.css`, `script.js`, and the `images/` folder).
3. Go to **Settings → Pages**.
4. Under "Source", pick branch `main` and folder `/ (root)`. Save.
5. After ~1 minute, your site is live at:
   `https://<your-username>.github.io/daamu/`

That's it. Share the link with Damu 💗 — from Achu.
